Frequently asked questions

Which devices are supported?
iPhone with iOS 16 or later. Android is in development — release date to be confirmed.
Is there a free trial?
Yes — a free 3-day trial of all features so you can try before you buy.
Do I need an account?
BetaSuite itself doesn't require an account, but you'll need a valid Apple ID / App Store account to purchase and download the app.
How do I delete my data?
Open BetaSuite and go to Settings → Data → Wipe all app data.
What is Confidential Mode?
Confidential Mode blocks cloud sync and cloud AI, keeping processing local-only where possible.
